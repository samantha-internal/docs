# conversations 

| API         | Description                                                                          |
|:------------|:-------------------------------------------------------------------------------------|
| `parse`     | Parse user input to understand intent and entities                                   |
| `chitchat`  | Conduct free-form chit-chat with your users                                          |
| `dialogtag` | Tag dialog utterances with [_dialog acts_](https://en.wikipedia.org/wiki/Dialog_act) |
